Project Bug Audit & Fix Guide

Frontend Issues

# File Problem Fix
1 axios.js JSON.parse(localStorage) may crash the app if malformed tokens exist. Wrap JSON.parse in try/catch.
2 AuthContext.jsx Login flow can leave the app in inconsistent state if profile request fails. Wrap login process in try/catch and rollback authentication.
3 Dashboard.jsx Dashboard uses mock data instead of backend API. Replace mock data with API request.
4 CompanyChat.jsx WebSocket does not reconnect when connection drops. Add reconnect logic in onclose.
5 CompanyChat.jsx Duplicate messages may appear due to optimistic UI update. Check message ID before adding to state.
6 EmployeeList.jsx Fetching all employees and filtering client-side causes performance issues. Implement backend pagination.
7 LeaveRequestList.jsx Grouping by employee name may merge records if names are identical. Group using employee_id instead.

Backend Issues

# File Problem Fix
1 consumers.py Manual JWT decoding bypasses authentication middleware. Use Channels AuthMiddlewareStack or JWT middleware.
2 consumers.py User permissions are checked only on connection. Recheck permissions inside receive().
3 EmployeeSerializer N+1 query when checking leave status. Use queryset annotation.
4 LeaveRequestViewSet Leave approval is not atomic, causing race conditions. Wrap approval logic in transaction.atomic().
5 EmployeeSerializer.create User and Employee creation not wrapped in transaction. Use transaction.atomic().

WebSocket Issues

# Component Problem Fix
1 CompanyChatConsumer Event types inconsistent between frontend and backend. Standardize event names.
2 Department Chat Typing indicators missing. Add typing broadcast events.
3 Department Chat Presence join/leave events missing. Broadcast presence events.

Architecture Improvements

# Area Improvement
1 Chat Attachments Handle attachments via REST API and broadcast using WebSocket.
2 Leave System Create leave balances automatically when a new leave type is created.
3 Messaging Add read receipts to department chat.
4 Leave Requests Add endpoint for employees to cancel pending requests.
